## Installation

J'ai choisi de réaliser ce projet en python (version 3.13), pour la simplicité ainsi que ma maitrise du langage, bien que la boucle for me soit insupportable. Sur windows, il faudra installer python sur le site officiel. Sur linux, il suffit d'entrer une ligne de commande selon les distribution.
* sur Ubuntu/Debian/Mint :
```bash
sudo apt install python3
```

Ayant travaillé sur une machine windows, il est possible que l'ajout du droit d'execution soit attendu sur une machine linux.
Pour ce faire, il suffit simplement d'entrer la commande :
```bash
chmod +x nom_du_fichier.py
```

Vous pouvez ensuite éxécuter le code avec la commande :
```bash
./nom_du_fichier.py
```
Je n'ai pas incu de shebang sur les fichiers, mais étant donné que tous les fichiers ont l'extension .py, cela devrait fonctionner.
